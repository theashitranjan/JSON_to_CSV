# 🧩 JSON to Tabular Converter

![Python](https://img.shields.io/badge/Python-3.7%2B-blue.svg)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)
![Status](https://img.shields.io/badge/status-active-brightgreen)

A simple Python tool to convert nested JSON data into a flat, tabular CSV format. This project helps extract structured, analyzable data from complex JSON objects.

---

## 📂 Use Case

This converter is ideal for:
- Transforming API responses
- Converting log data
- Preparing datasets for data analysis or visualization

---

## 🛠️ Features

- ✅ Handles deeply nested JSON structures
- ✅ Converts JSON arrays and objects into flat table rows
- ✅ Custom renaming of nested keys
- ✅ Export to CSV for use in Excel, Power BI, etc.
- ✅ Simple and minimal codebase
